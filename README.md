# QtSerialPortAssistant
Qt5下编写的串口网络调试助手
1. 支持串口调试通信。
2. 支持TCP Client/Server 调试。
3. 支持多窗口同时调试。
4. 支持简单的日志输出记录。
5. 支持Hex 和 ASCII 格式发送接收
6. 支持定时发送。
7. 支持保存打开配置文件。
8. 支持设置发送 接收颜色，时间格式等功能。
